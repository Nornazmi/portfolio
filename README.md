# Portfolio

A modern, responsive personal portfolio website built with Angular and TailwindCSS.

**[Live Demo](https://nornazmi.github.io/portfolio/)**

---

## About Me

I'm a passionate developer who loves building clean, efficient, and user-friendly web applications. This portfolio showcases my projects, skills, and professional journey.

## What This Project Offers

- **Home** – Introduction and quick overview
- **About** – Background, skills, and experience
- **Projects** – Showcase of my work with live links
- **Contact** – Ways to get in touch

## Tech Stack

| Category | Technologies |
|----------|-------------|
| Framework | Angular 18 |
| Styling | TailwindCSS |
| Icons | Lucide Angular |
| Deployment | GitHub Pages |

## Features

- Responsive design for all devices
- Dark mode toggle
- Smooth routing between sections
- Clean and modern UI

## Run Locally

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
